# CasaOS Restart Bot

Aplikasi sederhana untuk merestart service Telegram bot di CasaOS.

## Cara Install

1. Clone repository ini:
   ```bash
   git clone https://github.com/mknetid/restart-bot-telegram.git
   cd restart-bot-telegram
   
2. Jalankan aplikasi menggunakan Docker Compose:
   ```bash
   docker-compose up -d
   
3. Buka browser dan akses aplikasi di:
   http://<IP-CASAOS>:3233
   
4. Tekan tombol "Restart Bot" untuk merestart service telegram-bot.
